# ⚙️ Getting Started

## Clone Repository

```bash
git clone https://github.com/amolkanago/portfolio.git

cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

---

# 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
