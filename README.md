This portfolio highlights my journey as a software developer through carefully selected projects, professional experience, and technical skills.

Built with **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **Framer Motion**, it focuses on delivering a fast, accessible, and visually engaging user experience.

# ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/amolkanago/portfolio.git

cd portfolio
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start Development Server

```bash
npm run dev
```

Open **http://localhost:5173** in your browser.

---

# 📦 Build for Production

Create an optimized production build.

```bash
npm run build
```

Preview the production build locally.

```bash
npm run preview
```
